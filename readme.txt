=======================================
ZORG-Ω Application - Quick Start Guide
=======================================

Welcome to ZORG-Ω, your modular trade show data scraper!

How to start the program:
1. Make sure Node.js is installed on your computer.
2. Double-click the "install_requirements.bat" file to install any missing dependencies. (You generally only need to do this once).
3. Double-click the "Start.bat" file to run the application.

Using the tool:
1. A black terminal window will open, indicating the server is running. Leave this open!
2. Open your web browser (like Chrome, Edge, or Firefox) and go to: http://localhost:3000
3. Select the platform you want to scrape from the dropdown menu, fill in any required information (like URLs or IDs), and click "Start Extraction".
4. The live log window will show the progress of the scraper. 
5. Once complete, an Excel file with all standardized data will be outputted for you automatically.
6. To close the program, simply close the black terminal window.

Troubleshooting:
- If nothing loads at http://localhost:3000, ensure "Start.bat" was clicked and that the terminal hasn't reported an error.
- If an extraction fails halfway, check the Live Telemetry window for clues or simply restart the extraction.
